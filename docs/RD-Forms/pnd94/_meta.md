# ภ.ง.ด.94 — PIT Mid-year (ม.40(5)-(8))

## Identifiers
- **RD code:** ภ.ง.ด.94 (PND94 / P.N.D.94)
- **Thai name:** แบบแสดงรายการภาษีเงินได้บุคคลธรรมดา ครึ่งปี สำหรับผู้มีเงินได้ ม.40(5)(6)(7)(8)
- **English name:** Personal Income Tax Return — Mid-year (Rent, Professional, Contract, Business)
- **Purpose:** ยื่นภาษีกลางปี สำหรับบุคคลธรรมดาที่มีเงินได้ประเภท ค่าเช่า / วิชาชีพ / ค่าจ้างทำของ / ธุรกิจ

## Filing
- **Frequency:** Semi-annual (กลางปี)
- **Deadline:** สิ้น **ก.ย.** (เพื่อรายได้ ม.ค.–มิ.ย. ของปีเดียวกัน)
- **Where to file:** District Revenue Office / efiling.rd.go.th
- **Legal basis:** ม.56 ทวิ

## TEAS relevance
- **Module:** Outside scope (ผู้รับเงินยื่นเอง)
- **Note:** เฉพาะ freelancer / professional / landlord ที่มีรายได้ ม.40(5)-(8) — บริษัทไม่ใช้
