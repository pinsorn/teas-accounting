# ภ.ง.ด.91 — Personal Income Tax (Employment income only)

## Identifiers
- **RD code:** ภ.ง.ด.91 (PND91 / P.N.D.91)
- **Thai name:** แบบแสดงรายการภาษีเงินได้บุคคลธรรมดา สำหรับผู้มีเงินได้ ม.40(1) เพียงอย่างเดียว
- **English name:** Personal Income Tax Return — Employment Income Only
- **Purpose:** ยื่นภาษีเงินได้รายปี เฉพาะพนักงานที่มีเงินเดือนอย่างเดียว (ไม่มีรายได้อื่น)
