# ภ.พ.30.2 — VAT Return (Input Tax Adjustment by Revenue Ratio)

## Identifiers
- **RD code:** ภ.พ.30.2 (PP30.2)
- **Thai name:** แบบแสดงรายการภาษีมูลค่าเพิ่ม กรณีปรับปรุงภาษีซื้อที่เฉลี่ยตามส่วนของรายได้
- **English name:** VAT Return — Input Tax Apportionment Adjustment (by revenue ratio)
- **Purpose:** ปรับปรุง Input VAT ของกิจการที่มีทั้งกิจการ VAT + กิจการ exempt — เฉลี่ยตามสัดส่วนรายได้
- **Version (latest):** ฉบับ 2568

## Filing
- **Frequency:** Annual — ภายในเดือนถัดไปหลังสิ้นปี (ปรับปรุง Input VAT ที่ apportionment ทั้งปี)
- **Deadline:** 25 ของเดือนถัดไป (e-filing)
- **Where to file:** efiling.rd.go.th
- **Legal basis:** ประมวลรัษฎากร ม.82/5 + ม.82/6

## TEAS relevance
- **Module:** Tax / Year-end adjustment
- **Trigger:** กิจการที่มีทั้ง VAT business + Exempt business — ต้อง apportion Input VAT
- **Use case:** บริษัทที่มีรายได้ ม.81 (exempt) ผสม VAT 7% — ตัวอย่าง: บริษัทขายสินค้า VAT + ให้เช่าอสังหา exempt
