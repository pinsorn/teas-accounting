# ภ.ง.ด.55 — ภาษีเงินได้มูลนิธิ/สมาคม

## Identifiers
- **RD code:** ภ.ง.ด.55 (PND55 / P.N.D.55)
- **Thai name:** แบบแสดงรายการภาษีเงินได้มูลนิธิหรือสมาคม หรือกองทุนรวม
- **English name:** Income Tax Return — Foundation / Association / Mutual Fund
- **Purpose:** ยื่นภาษีเงินได้สำหรับมูลนิธิ / สมาคม / กองทุนรวม (ไม่ใช่บริษัท)

## Filing
- **Frequency:** Annual
- **Deadline:** ภายใน 150 วันจากสิ้นรอบบัญชี (เหมือน ภ.ง.ด.50)
- **Where to file:** District Revenue Office / efiling.rd.go.th
- **Legal basis:** ม.68 + ม.69

## TEAS relevance
- **Module:** Tax (เฉพาะ entity type = Foundation/Association)
- **Trigger:** ถ้าบริษัทใน TEAS เป็นมูลนิธิ
- **Rates:**
  - 2% ของรายได้ก่อนหักรายจ่าย (default)
  - 10% ของรายได้สุทธิ (เฉพาะที่ระบุ)
  - มูลนิธิที่ได้รับอนุมัติยกเว้น → ไม่ต้องเสียภาษี (แต่ยังยื่น)
