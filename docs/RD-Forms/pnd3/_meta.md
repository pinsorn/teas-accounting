# ภ.ง.ด.3 — WHT จ่ายให้บุคคลธรรมดา ม.40(5)-(8) รายเดือน

## Identifiers
- **RD code:** ภ.ง.ด.3 (PND3 / P.N.D.3)
- **Thai name:** แบบยื่นรายการภาษีเงินได้หัก ณ ที่จ่ายตามมาตรา 52 และมาตรา 59 กรณีหักภาษี ณ ที่จ่ายตามมาตรา 50(3)(4)(5) สำหรับเงินได้พึงประเมินตามมาตรา 40(5)(6)(7)(8)
- **English name:** Withholding Income Tax Return — Payment to Individual Contractors / Services
- **Purpose:** ยื่น WHT ที่หักจากการจ่ายค่าเช่า / ค่าวิชาชีพ / ค่าจ้างทำของ / ค่าบริการ ให้บุคคลธรรมดา (freelancer / professional)

## Filing
- **Frequency:** Monthly
- **Deadline:**
  - Paper: **7** ของเดือนถัดไป
  - e-Filing: **15** ของเดือนถัดไป
- **Where to file:** District Revenue Office / efiling.rd.go.th
- **Legal basis:** ม.52 + ม.59 + ม.50(3)(4)(5) + ม.48 ทวิ · เงินได้ตาม ม.40(5)(6)(7)(8)

## TEAS relevance
- **Module:** AP / Tax
- **Trigger:** Payment Voucher (PV) ที่จ่ายให้ vendor ที่เป็น บุคคลธรรมดา (`vendor_type = Individual` / `is_juristic = false`)
- **Threshold:** ยอด ≥ 1,000 บาท/ครั้ง
- **Rates per §2.2 of Tax-Reference-TH.md:**
  - ม.40(5) ค่าเช่า: **5%**
  - ม.40(6) ค่าวิชาชีพ: **3%**
  - ม.40(7) ค่าจ้างทำของ: **3%**
  - ม.40(8) ค่าบริการทั่วไป: 3%
- **Pair with:** ออก 50 ทวิ ให้ freelancer ทุกคน
