# ป.ป.01 — แบบคำขอแก้ไขข้อมูลภาษีเงินได้หัก ณ ที่จ่าย

## Identifiers
- **RD code:** ป.ป.01 (POR.POR.01)
- **Thai name:** แบบคำขอแก้ไขข้อมูลภาษีเงินได้หัก ณ ที่จ่าย
- **English name:** WHT Information Correction Request Form
- **Purpose:** ขอแก้ไขข้อมูลในแบบยื่น ภ.ง.ด.1/2/3/53/54/55 หรือใบแนบที่ยื่นไปแล้ว (กรณีพบข้อผิดพลาด)

## Filing
- **Frequency:** ad-hoc (เมื่อพบข้อผิด)
- **Deadline:** ภายในกำหนดเวลาที่อธิบดีกำหนด (ปกติภายใน 1 ปี)
- **Where to file:** District Revenue Office
- **Legal basis:** ตามประกาศอธิบดีฯ

## TEAS relevance
- **Module:** Tax / Correction workflow
- **Trigger:** หลังจาก post WHT แล้ว → พบ error ใน 50 ทวิ ที่ออกไป (เช่น TIN ผู้รับผิด, จำนวนเงินผิด)
- **TEAS Phase 1:** out of scope (correction workflow ทำ manual ก่อน)
- **TEAS Phase 2+:** ระบบควรช่วย generate ป.ป.01 + ใบแนบที่ระบุ original + corrected
