# ภ.ง.ด.51 Page-2 (Computation Worksheet) Fill — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended)
> or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Fill page 2 (การคำนวณภาษี worksheet) of the official ภ.ง.ด.51 AcroForm for the **Method A
(ม.67ทวิ(1)), clean first-filing** case, behind a user **attestation** guard, and render it in the same
PDF as page 1.

**Architecture:** (1) Make `RdAcroFormFiller.Render` **page-aware** (overlay each field on its own widget
page; single-page forms keep page-0 behaviour exactly → zero regression). (2) Extend `Pnd51Model` +
`Pnd51FilingService` with the worksheet values (derived from the existing CIT engine + H1 P&L) and an
**attestation** that *throws* unless the filer confirms: first filing · no loss carry-forward · no
exemption · no rate-reduction. (3) Map the page-2 method/currency/sign/rate **radios** (visually
confirmed). (4) Fill the derivable amount boxes; leave the unknowable ones blank only when attested 0.

**Tech Stack:** C#/.NET 10, PdfSharp (geometry+composite), QuestPDF/Skia (Thai overlay), xUnit+FluentAssertions
(BE), Next.js 15 + RHF/Zod (FE), pymupdf (visual verification, dev-only). Source of truth:
`docs/superpowers/specs/pnd51-page2-map.md`. Decisions locked with Ham: **path 1 (minimal honest fill)**,
**Attestation guard**, **page-aware Render (general, no pnd51 branch)**, **box-32 rate via existing `schedule`**.

## Compliance guardrails (read before coding — §4/§11)

- **Omission asserts zero.** On this form, a blank amount box = "the value is 0". So page-2 may be filled
  **only** when boxes 55 (loss c/f), 56 (exemption), 34 (rate-reduction) are *attested* zero, the filing is
  **not amended**, and there is **no ม.27/67ตรี surcharge**. The guard below enforces this; it MUST throw,
  never silently default. Cite `// ภ.ง.ด.51 §4 attestation` on the guard + its test.
- **Never guess a radio.** Each page-2 radio tick in Task 1 is confirmed against a rasterised, index-labelled
  page-2 image before it is used in Task 5. An un-ticked method box = ambiguous worksheet = not audit-clean.
- **Method A only.** Method B (ม.67ทวิ(2), boxes 61-66) and gross-receipts (66.1) stay blank in v1.

**Design:** every overlay primitive gains a **page index**. `ReadFieldRects` records each terminal
widget's page (scan each `doc.Pages[i].Elements["/Annots"]`, map widget→i; fallback: page 0) and the

```csharp
namespace Accounting.Api.Tests.Pdf;

public sealed class RdAcroFormFillerMultiPageTests
{
    // ภ.ง.ด.51 is the only 2-page template; page-2 fields must land on page index 1.
    [Fact]
    public void Page2_field_renders_onto_page2_not_page1()
    {
        var tpl = Templates.Load("pnd51_main.pdf");              // helper added in Step 3
        var before = RdAcroFormFiller.Render(tpl, new[] { new RdField("Text1.2", "PAGE1NAME") },
            System.Array.Empty<RdRadio>(), null);
        var after = RdAcroFormFiller.Render(tpl, new[] {
            new RdField("Text1.2", "PAGE1NAME"),
            new RdField("Text4.3", "999999") }, System.Array.Empty<RdRadio>(), Pnd51Cells.Map);

        // page-2 content stream must grow when a page-2 field is added; page-1 size ~unchanged.
        var (p1Before, p2Before) = ContentLengths(before);
        var (p1After,  p2After)  = ContentLengths(after);
        p2After.Should().BeGreaterThan(p2Before + 50, "Text4.3 must draw onto page 2");
        p1After.Should().BeCloseTo(p1Before, 50, "page-1 overlay must be unchanged");
    }

    private static (int p1, int p2) ContentLengths(byte[] pdf)
    {
        using var ms = new MemoryStream(pdf);
        var doc = PdfReader.Open(ms, PdfDocumentOpenMode.ReadOnly);
        return (PageContentLen(doc.Pages[0]), PageContentLen(doc.Pages[1]));
    }
    private static int PageContentLen(PdfSharp.Pdf.PdfPage pg)
    {
        var len = 0;
        foreach (var c in pg.Contents.Elements) len += ((PdfSharp.Pdf.PdfReference)c).Value is PdfSharp.Pdf.PdfDictionary d && d.Stream != null ? d.Stream.Value.Length : 0;
        return len;
    }
}
```
