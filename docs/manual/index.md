# คู่มือการใช้งาน TEAS

**Thailand Enterprise Accounting System** — ระบบบัญชีสำหรับองค์กรไทย
ที่รองรับภาษีมูลค่าเพิ่มและ e-Tax ตั้งแต่วันแรก

> เนื้อหาคู่มือนี้ส่วนที่เป็นหน้าจอจริงสร้างอัตโนมัติจากการบันทึกหน้าจอของระบบ
> (Playwright headless capture) — ภาพและขั้นตอนตรงกับแอปที่ใช้งานจริง ไม่ drift.

## วิธีอ่านคู่มือ

อ่านตามลำดับบท: เริ่มจาก **บทที่ 0 (ติดตั้ง)** สำหรับผู้ดูแลระบบ → **บทที่ 1
(เริ่มต้นใช้งาน)** สำหรับผู้ใช้ทุกคน → บทตามโมดูลที่ใช้งาน. แต่ละหัวข้อเป็น
**step-by-step** พร้อมภาพหน้าจอประกอบทุกขั้น.

## แผนผังคู่มือ (ทุกโมดูล)

| บท | เรื่อง | สถานะ |
|---|---|---|
| **0** | **ติดตั้งระบบ + ตั้งผู้ดูแล** — prerequisites, install, ฐานข้อมูลตั้งต้น, Super Admin คนแรก, สร้างบริษัท + ตั้งค่าภาษี, สร้าง role/ผู้ใช้ | ✅ พร้อม |
| **1** | **เริ่มต้นใช้งาน** — เข้าสู่ระบบ, แดชบอร์ด, สลับภาษา, ออกจากระบบ | ✅ พร้อม |
| **2** | **ตั้งค่าระบบ** — ข้อมูลบริษัท, หน่วยธุรกิจ, สินค้า/บริการ, ประเภทหัก ณ ที่จ่าย, API keys | ✅ พร้อม |
| — | **สิทธิ์การมองเห็น (RBAC UI Guide)** — ใครเห็น/ใช้เมนูและปุ่มใดได้บ้าง (ทุก role × บริษัท VAT/ไม่ VAT) | ✅ พร้อม |
| **3** | **ข้อมูลหลัก** — ลูกค้า, ผู้ขาย, ผังบัญชี, หมวดค่าใช้จ่าย, พนักงาน | 🚧 ถัดไป |
| **4** | **งานขาย** — ใบเสนอราคา → ใบสั่งขาย → ใบส่งของ → ใบแจ้งหนี้ → ใบกำกับภาษี → ใบเสร็จ, ใบลดหนี้/เพิ่มหนี้ | 🚧 ถัดไป |
| **5** | **งานซื้อ** — ใบสั่งซื้อ → ใบสำคัญจ่าย → ใบกำกับภาษีซื้อ → หนังสือรับรองหัก ณ ที่จ่าย (50 ทวิ) | 🚧 ถัดไป |
| **6** | **เงินเดือน** — พนักงาน, รอบเงินเดือน, ภ.ง.ด.1, ประกันสังคม | 🚧 ถัดไป |
| **7** | **ภาษีและการยื่น** — ภ.พ.30, ภ.ง.ด.3/53/54, สรุปภาษีรายเดือน | 🚧 ถัดไป |
| **8** | **รายงาน** — งบทดลอง, กำไรขาดทุน, ช่องว่างเลขเอกสาร, อายุหนี้ | 🚧 ถัดไป |
| **9** | **e-Tax** — ใบกำกับภาษีอิเล็กทรอนิกส์ (XAdES) + ส่งอีเมล + cc สรรพากร | 🚧 ถัดไป |

> บทที่มีสถานะ 🚧 จะทยอยเพิ่มแบบ step-by-step พร้อมภาพหน้าจอจริงในรอบถัดไป.
