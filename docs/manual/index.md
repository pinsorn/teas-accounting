# คู่มือการใช้งาน TEAS

**Thailand Enterprise Accounting System** — ระบบบัญชีสำหรับองค์กรไทย
ที่รองรับภาษีมูลค่าเพิ่มและ e-Tax ตั้งแต่วันแรก

> เวอร์ชัน **0.5** — นำร่องบทที่ 1–2 (เริ่มต้นใช้งาน + ตั้งค่าระบบ)
> เนื้อหาในคู่มือนี้สร้างอัตโนมัติจากการบันทึกหน้าจอจริงของระบบ
> (Playwright headless capture) — ภาพและขั้นตอนตรงกับแอปที่ใช้งานจริง

## โครงสร้างคู่มือ

- **บทที่ 1 — เริ่มต้นใช้งาน**: เข้าสู่ระบบ, แดชบอร์ด, สลับภาษา, ออกจากระบบ
- **บทที่ 2 — ตั้งค่าระบบ**: หน่วยธุรกิจ, สินค้า/บริการ, ประเภทหัก ณ ที่จ่าย,
  API keys, ข้อมูลบริษัท

บทเพิ่มเติม (ขาย/ซื้อ/รายงาน) จะทยอยเพิ่มในเวอร์ชันถัดไป
