## 04.03 — ออกใบแจ้งหนี้ / ใบวางบิล

> **เงื่อนไขก่อนใช้งาน:** login admin (สิทธิ์ billing_note) · มีลูกค้า + หน่วยธุรกิจ ในระบบ

ใบแจ้งหนี้ / ใบวางบิล (Billing Note) คือเอกสาร **เรียกเก็บเงิน** ที่ส่งให้ลูกค้าเพื่อ
แจ้งยอดที่ต้องชำระและกำหนดวันครบกำหนด.

**ใช้งานได้ 2 แบบ:**

1. **รวมใบกำกับภาษีหลายใบ** — กิจการที่ส่งของหลายครั้งในเดือน แล้ววางบิลรวมครั้งเดียว
   สามารถเลือกใบกำกับภาษีที่ออกไว้แล้วมารวมในใบแจ้งหนี้ใบเดียว.
2. **ออกพร้อมรายการของตัวเอง** — กรอกรายการสินค้า/บริการลงในใบแจ้งหนี้โดยตรง
   (เหมือนใบเสนอราคา) ตามที่บทนี้แสดง.

เมื่อลูกค้าชำระเงิน จะออก **ใบเสร็จรับเงิน** อ้างอิงใบแจ้งหนี้/ใบกำกับภาษีนี้ (ดู 04.05).
มีช่อง "วันที่ครบกำหนด" (ค่าเริ่มต้น +30 วัน) เพื่อติดตามลูกหนี้.
